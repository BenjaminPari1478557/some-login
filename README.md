# some-login
## Formulario de Registro
Para levantar el proyecto primero clone el repositorio
y levante la infraestructura con:

docker-compose up --build

## Para acceder a la aplicacion, acceda desde estas URLs:

- Frontend: http://localhost:4200

- RabbitMQ Management: http://localhost:15672 (guest/guest)

Los puertos de los servicios esan declarados en el .env.example de ejemplo que deben estar en un .env
